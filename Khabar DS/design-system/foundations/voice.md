# Voice — Khabar Design System

---

## The editorial voice in one sentence

Khabar writes the way a trusted senior editor speaks: plainly, precisely, without embellishment — the voice of someone who has reported for 20 years and earns trust by being right, not by being loud.

---

## Seven voice rules

1. **Facts before adjectives.** Write "₹4,200 करोड़ की घोषणा" not "बड़ी राशि की घोषणा।"
2. **Sentence case, always.** Headlines are not title-cased. No टाइटल केस।
3. **Short sentences.** 15–20 words max per sentence in digital copy. Split ruthlessly.
4. **Name the reader's world.** Use their nouns — संसद, मंत्री, मैदान, बाज़ार, किसान, छात्र — not our nouns (प्लेटफॉर्म, सॉल्यूशन, इकोसिस्टम).
5. **Every claim has a number or a name.** "कई लोग" → cite how many. "सरकार ने कहा" → which minister, when.
6. **No hype vocabulary.** See banned words below.
7. **Second person for newsletters and features.** आप (respectful singular) — never the plural "हम सब" condescension.

---

## 3-second feeling test

Before publishing any piece of copy, ask: does this make the reader feel *informed* or *manipulated*? Informed = publish. Manipulated = rewrite.

## Tone by context

| Surface | Register |
|---|---|
| Breaking news headline | Neutral, fast, exact — no adjectives |
| News article lede | Who, what, when, where in one sentence |
| Analysis / explainer | Calm, contextual — "यहाँ समझें" |
| Opinion column header | Authoritative, first-person is allowed |
| Sports report | Energetic, but never hyperbolic — cite the score |
| Election results | Data-first, then context |
| Business / markets | Precise, numbers lead |
| Entertainment news | Lighter tone — still factual, not tabloid |
| Newsletter subject line | Direct, specific — no "don't miss this!" |
| Error messages | Plain, helpful, non-technical |
| 404 page | Calm, one-line redirect — not playful |
| About page | Institutional, modest, direct |
| Footer legal | Plain English (or Hindi) — no legalese |
| Push notification | Headline only, no emoji |

## Grammar & punctuation house style

- **Dates:** DD Month YYYY — 15 अगस्त 2024
- **Numbers:** Lakh/crore system — ₹4.2 करोड़, not ₹42 million
- **Currency:** ₹ symbol, no space before number
- **Time:** 24-hour in data contexts, 12-hour AM/PM for readers
- **Quotes:** "…" for direct quotes. No smart quotes in digital
- **Dashes:** — em dash for parentheticals, - hyphen for compound words
- **Ellipsis:** Only in direct quotes where text is omitted. Never for suspense.
- **Exclamation marks:** Only in sports scores / match-result headlines. Limit 1 per article.
- **Devanagari numerals:** Use Arabic numerals (1, 2, 3) in all contexts for readability

## Banned vocabulary

| Banned | Why | Use instead |
|---|---|---|
| सनसनीखेज / shocking | Editorialises | Report what happened |
| भारी / huge / massive | Vague | Give the actual number |
| धमाकेदार | Tabloid | Name the event |
| ऐतिहासिक / historic | Overused, claim needs evidence | Cite what makes it historic |
| खुलासा हुआ / revealed | Implies hidden conspiracy | "रिपोर्ट में सामने आया" |
| वायरल | Ephemeral, rarely news | Specify platform + reach |
| क्रांति / revolution | Hype | Describe the actual change |
| गेम-चेंजर | Corporate jargon | State the specific impact |

## What to do when there's nothing specific to say

If a story has no confirmed numbers, no named sources, and no verified outcome — **don't publish it yet.** Khabar waits for a second source before breaking a story. When publishing with incomplete information, say so explicitly: "विवरण अभी सामने आ रहे हैं।" *(Details are still emerging.)*
