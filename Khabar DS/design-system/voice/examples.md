# Voice Examples — Khabar Design System

Do/don't pairs for every major surface. The rule: state facts, name numbers, skip adjectives.

---

## Homepage hero headline

| | Example |
|---|---|
| ✓ Do | सरकार ने ₹1.2 लाख करोड़ का कृषि बजट पास किया |
| ✗ Don't | बड़ी खबर! सरकार ने किसानों को दिया जबरदस्त तोहफा |
| ✓ Do | भारत ने ऑस्ट्रेलिया को 6 विकेट से हराया, सीरीज 3-1 से जीती |

## Anti-pattern checklist

Before publishing any copy, check:

- [ ] क्या किसी हेडलाइन में "सनसनीखेज", "धमाकेदार", "जबरदस्त", "ऐतिहासिक" है? → हटाएं
- [ ] क्या कोई दावा बिना संख्या या नाम के है? → जोड़ें
- [ ] क्या "ब्रेकिंग" का इस्तेमाल सिर्फ पुरानी खबर पर है? → हटाएं
- [ ] क्या CTA में exclamation mark है? → हटाएं
- [ ] क्या subject line में emoji है? → हटाएं
- [ ] क्या title case का इस्तेमाल हुआ है? → sentence case में बदलें
